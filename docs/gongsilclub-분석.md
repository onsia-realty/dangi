## 1. NEW 공실클럽 (www.gongsilclub.co.kr)

### 전체 메뉴 구조
매물관리 / 단독중개 / 공동중개 / 매물찾습니다 / **등기·대장** / 채팅 / 중개사무소 / 커뮤니티 / 상품&결제

### 매물 상세 표현방식 (article/?id=14253 기준)
헤더 영역:
- 매물번호 + 매물종류·층수 요약 (예: "중소형사무실, B1층 / 7층")
- 가격: `월세 5,000만원/450만원 / 기준가 5억원` — **기준가(환산가) 병기**
- 면적: `계약/전용 면적 : 217.12㎡/190㎡ (전용률 88%)` — **전용률 자동 계산 표시**
- 등록 중개사무소 + 연락처 2개 + **채팅하기** 버튼
- 유틸: 매물 상세 인쇄(사진/지도/연락처 포함 옵션), 수정신고

정보 테이블 (라벨-값 2열, 없는 값은 "-"):
매물 특징(한줄) / 소재지 / 건축물 용도 / **위반 건축물 여부** / 융자금 / 월 관리비 /
입주 가능일(협의가능 표시) / 계약·전용 면적 / 해당 층·총 층 / 주된 출입구 방향 /
화장실 수 / 총 주차대수 / 총 사무실수 / 주차 여부 / 난방 방식·연료 / **사용승인일** /
옵션(태그 나열) / 상세 설명(불릿) / 등록일 / 수정일

하단: 지도(카카오) + 사진 슬라이드 + 책임 면책 문구

### 매물 등록 폼 (6단계 원페이지 스크롤 위저드)
0. **광고 채널 선택**: 네이버+공동중개 / 네이버 / 공동중개 (채널별 필수값 다름)
1. **위치**: 매물종류(16종) → 시/도·시/군/구·읍/면/동 → 단지명/면적(DB 선택식, 전용률 자동) → 동/호 (미등기 체크 지원) → 건축물 주용도
2. **거래정보**: 거래종류(매매/전세/월세/**단기**) → 가격 (만원 입력 시 원 단위 환산 실시간 표시) → 융자금(과태료 경고) → 관리비 부과방식 → **임차권등기명령 여부** → 입주가능일(협의가능)
3. **구조**: 해당층/총층(노출방식 선택) → 방향(기준+방향) → 방/화장실 수 → 현관/내부 구조 → 주차대수(네이버 단지정보 자동 입력) → 주차 여부
4. **시설정보**: 냉방/생활/보안/기타 시설 체크박스 태그
5. **상세정보**: 매물 특징(40자, 리스트 노출용) / 상세설명(1000자) / 사진(20장, 장당 5MB, 총 100MB) / **중개사 메모(비공개)**
6. **중개사무소**: 노출 연락처 선택(대표번호/휴대폰) / 주소 노출(지번/도로명 선택)

특징: 단계별 앵커 네비게이션, 필수값 `*` + "*필수 정보입니다" 인라인 검증, 법적 경고(과태료) 인라인 노출, 툴팁(ⓘ) 안내

### 등기·대장 (건축물대장/등기부등본 통합 조회) ★핵심 기능
- 흐름: `+ 신규 열람` → 공부 종류(건축물대장/등기부등본) → 건축물 종류(일반/집합) → 지번 주소 전체 입력 → 조회
- 열람 내역 관리: 열람번호 / 일시 / 열람자 / 공부종류 / 부동산구분 / 고유번호 / 소재지 + 기간·종류·통합검색 필터
- 조회·출력·매물 첨부까지 사이트 안에서 원스톱, 열람 내역으로 자료 제출 대응
- 매물 등록 시 건축물대장·등기부등본 연동으로 주소/용도/사용승인일 등 자동 채움 (네이버 광고 소개 문구에서 확인)

### 기타 참조 포인트
- 동시접속 1세션 정책 (다른 PC 로그인 시 강제 로그아웃 확인창)
- 접속 IP 상시 표시 (보안/증빙)
- 매물 리스트 카드: `단지명·층 / 가격 / 매물종류 / 면적` 4줄 압축형
- 지역 필터: 시/도→구→동 드릴다운 + 구별 매물 수 뱃지
- 같은 매물 재광고 금지 정책 (광고 종료 매물은 신규 등록 필요)

## 2. 구 공실클럽 (gc3.gongsilclub.com/v4)

### 메뉴
공실매물(원룸/투룸+/오피스텔빌라/사무실/상가/아파트/토지) / 공동중개 / 매매정보 / 매물셀프코너 / 커뮤니티 / 고객센터 / MY결제 / 매물의뢰

### 공실매물 검색 UX (참조 가치 높음)
- **블록지도**: 동 단위 + 지하철역 아이콘이 찍힌 커스텀 블록 지도에서 블록 클릭 → 해당 구역 매물 (일반지도/주소검색 전환 가능)
- 필터: 거래구분 / 룸갯수(1~5+, 0.5 단위) / 룸형태(오픈형·분리형·복층형·원룸원거실·세미분리형) / 층구분(지상·지하·반지하·옥탑·단독) / 준공년도 / 거실크기 / 주차대수
- **추가필터에 "단기임대" 존재** — 우리 서비스와 직접 연관
- 기타: 건물·내부사진/동영상 유무, 현재공실, 풀옵션 필터, E/V, 금액네고, 전세대출가능, 애완동물가능
- 가격: 기준가/보증금/월세 범위 + **관리비 포함 검색**
- 면적: ㎡/평 단위 토글
- 편의: 현재필터를 첫화면으로 저장, 선택매물 일괄 인쇄(특이사항 포함 옵션), 관심매물 담기

### 구 사이트의 공부(公簿) 처리
- 인터넷등기소(iros.go.kr), 서울부동산정보(land.seoul.go.kr) **외부 링크만 제공** → NEW에서 내장 조회로 진화한 것

## 3. BOOIN(단기임대)에 가져올 기능 우선순위 제안

1. **건축물대장/등기부등본 조회 연동** — 정부 API(건축HUB 건축물대장, 대법원 인터넷등기소는 유료 API) 기반. 매물 등록 시 주소 입력 → 용도/사용승인일/위반건축물 여부 자동 채움 + 열람 내역 관리
2. **매물 상세 표준 레이아웃** — 라벨-값 테이블 + 위반건축물/사용승인일/관리비 등 법정 명시사항 (중개대상물 표시·광고 규정 대응)
3. **단계식 매물 등록 위저드** — 위치→거래정보→구조→시설→상세→연락처, 만원→원 환산 표시, 인라인 필수값 검증
4. **단기임대 특화 필터** — 구 사이트의 "단기임대·현재공실·풀옵션" 필터를 1급 필터로 승격 (입주가능일 캘린더 기반)
5. **관리비 포함 가격 검색** — 단기임대는 관리비 비중이 커서 실부담액 기준 검색이 유효
6. **블록지도식 지역 탐색** — 동/역 단위 블록 선택 (네이버 지도 위 오버레이로 현대화)
7. **매물 인쇄/PDF + 중개사 비공개 메모** — 중개사 실무 편의
